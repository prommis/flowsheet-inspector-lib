# Flowsheet Inspector Library Documentation

This site contains documentation for `idaes-fi`, the Python package that
provides library modules and API support for the IDAES Flowsheet Inspector user
interface.

## Getting Started

Install the package in editable mode with development dependencies:

```bash
python -m pip install -e .[dev]
```

## Package Layout

- PyPI package name: `idaes-fi`
- Import package: `idaes_fi`
- Source layout: `src/idaes_fi`
